CSS Combinators Ejemplo de Sintaxis
Decendant Selector Selecciona todos los descendientes que cierto elemento div p. Todos los elementos p que estén dentro de div tendrán el diseño
Child Selector Selecciona a todos los hijos inmediatos de un cierto elemento div > p. Todos los p que sean hijo inmediato de div
Adjacent Sibling Selector Selecciona al elemento siguiente en el mismo nivel que cierto elemento. Deben tener mismo padre. div + p. El elemento p que este justo algún div
General Sibling Selector Selecciona a los elementos del mismo nivel que cierto elemento. Deben tener mismo padre. div ~ p. Los elementos p que esten justo después de div
